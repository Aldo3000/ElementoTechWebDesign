# ElementoTechWebDesign
Este proyecto consiste en un rediseño conceptual de un sitio web de un negocio local, desarrollado de manera independiente y no oficial, con el objetivo de practicar y demostrar mis habilidades en desarrollo front-end utilizando HTML y CSS.  El diseño fue realizado únicamente con fines educativos y de portafolio.

El proyecto puede usarse para mejorar la atraccion de clientes, instalar un pixel con la plataforma de anuncios de META para generar conversiones o leads mas calificados para ventas en WhatsApp. Anuncio -> Landing PAge -> WhatsApp. El comercio se observa mucho que sus ventas son en base por WhatsApp.
